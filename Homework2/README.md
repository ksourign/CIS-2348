Password Modifier - takes a simple password and makes it stronger by replacing characters using the key below
Appending "q*s" to the end of the input string.

i becomes !
a becomes @
m becomes M
B becomes 8
o becomes .
Ex: If the input is:
mypassword

the output is:
Myp@ssw.rdq*s
